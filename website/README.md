# Sri Anantha Transocean LLP Website

Static single-page site inspired by premium export partners.

## Structure
- `index.html` – content and sections (hero with imagery, products, services, process, CTA).
- `styles.css` – light palette theme, layout, responsiveness.
- `script.js` – mobile nav toggle and smooth scrolling.
- `assets/` – place images, fonts, and other static files if needed.

## Quick start
Open `index.html` in a browser. Update copy, contact info, or product categories to match your business.
